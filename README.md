### Here are the steps to run the application in your local environment.

1 . Clone the repository using this command.

```
git clone https://github.com/Arridev-Production/Envato-Backend.git
```

2. Navigate to the folder.

```
cd Envato-Backend
```

3 . Run the command to install all the dependencies.

```
npm install
```

3 . Rename the `template.env` file to `.env` and write all the given variables value.

5 . Then run this command below to start the server.

```
npm run dev
```

#### Now go the this address of your browser `http://localhost://5000` to see the output.
